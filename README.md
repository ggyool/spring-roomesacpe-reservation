### 기능 요구 사항
- [ ] 테마 관리 기능 추가
- [ ] 테마별 스케줄 관리 기능 추가
- [ ] 추가된 테마와 스케줄에 따라 기존에 구현된 예약을 리팩터링

- [ ] 계층을 고려한 리팩터링을 한다.
